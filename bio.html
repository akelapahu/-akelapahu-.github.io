About Me

Saya adalah seorang siswa yang sekarang sedang bersekolah di Sekolah Menengah Kejuruan - SMAK Bogor. Saat ini saya masih 'ditempa' untuk menjadi seorang Analis Kimia yang handal di kemudian hari setelah saya lulus dari sekolah ini. Di sini, saya mendapatkan kesempatan untuk dapat belajar coding. Maka saya membuat halaman ini sebagai uji coba pengetahuan saya dan juga agar masyarakat dapat lebih mengenal saya. Dapat dilihat halaman ini masih jauh dari kata sempurna dan saya masih mempelajari coding lebih lanjut lagi, sehingga saya dapat membuat halaman yang lebih baik lagi suatu saat nanti.

Hobi saya diantaranya adalah membaca buku, mengoperasikan komputer, bermain game (terutama DOTA 2), dan juga bermain bulutangkis. Makanan kesukaan saya adalah daging rendang. Saya sangat suka mengotak-atik komputer. Saya selalu penasaran bagaimana sebuah komputer dapat bekerja dengan baik, bagaimana suatu aplikasi dapat dijalankan, bagaimana aplikasi tersebut dibuat dan masih banyak lagi hal-hal yang berkaitan dengan komputer yang sering saya cari tau mengapa dapat terjadi. Saat mengoperasikan komputer saya mempunyai aplikasi-aplikasi khusus yang membantu saya dalam mencari suatu hal yang berhubungan dengan sistem. Sedangkan untuk di bidang olahraga, saya rasa bulutangkis merupakan olahraga yang sesuai dengan saya dan saya menyukainya. Saya sangat bersemangat apabila saya mendapatkan kesempatan untuk bermain bulutangkis. Saya juga suka membaca buku, terutama buku cerita dalam bahasa Inggris sehingga saya dapat mempelajari tentang bahasa Inggris lebih baik lagi.

Saat mengotak-atik komputer, saya juga kadang suka mengedit sesuatu, seperti gambar, video, ataupun membuat animasi. Saya juga suka bermain game yang ada di PC ketika saya sedang bosan. Game yang paling saya sukai yaitu game DOTA 2, dimana dalam satu pertandingan terdapat 10 pemain dengan 5 pemain di masing-masing tim, dan mereka terlibat dalam peperangan yang memerlukan strategi agar dapat memenangkannya. Suatu ketika saat saya sedang menonton salah satu pertandingannya di Jakarta, saya mendapat kesempatan untuk berfoto dengan salah satu pemain profesional DOTA 2 dari sebuah tim bernama Team Liquid. Pemain ini bernama Kuro "Kuroky" Salehi Takashomi. Karena saya suka mengutak-atik komputer, maka saya edit foto bersama Kuro sehingga tampak lebih baik lagi.
